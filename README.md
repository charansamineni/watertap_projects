# watertap_spacer_value

Custom WaterTAP extensions for modeling and analyzing reverse osmosis spacers.

## Overview

This package extends the WaterTAP framework by adding custom unit models and utilities focused on spacer geometry and performance in reverse osmosis systems.

## Installation

Make sure you have Python 3.8+ and a conda environment set up.

```bash
# Activate your environment
conda activate watertap_local

# Install WaterTAP and dependencies
pip install idaes watertap

# Install your custom package in editable mode
pip install -e .
